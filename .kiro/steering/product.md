# Product Overview

This is an Apple iPhone 15 inspired marketing website clone featuring modern UI/UX design with interactive 3D models and smooth animations.

## Key Features

- Interactive 3D iPhone models with color variants
- Smooth scroll-based animations using GSAP
- Video carousel showcasing product features
- Responsive design with dark theme
- Modern landing page sections: Hero, Highlights, Features, How It Works

## Purpose

A showcase project demonstrating advanced web animation techniques and 3D rendering capabilities for product marketing websites.
